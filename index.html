CS 4300




This website was created to function as a learning tool. The links provide explanations and tutorials for HTML, CSS, and JavaScript. The data provided on the site is meant to give an entry point and overview of the subjects covered in CS 4300; the information can function as an accessible beginning for an individual interested in the class, and or website creation.



HTML, Hypertext Markup Language is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.
Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.This separation can improve content accessibility.
JavaScript is a scripting language which is used to enhance the functionality and appearance of web pages. Javascript is more complicated than HTML and CSS but carries greater functionalities.
